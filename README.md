# メタモンの富士山冒険 🟣🔫💦

Pang(パンブラザーズ)風の2Dブラウザゲーム。富士山が見える海を背景に、メタモン風キャラクターが水鉄砲でボールを撃ち落とそう！

## ライブデモ

**[今すぐプレイ](https://mihyangcho.github.io/2DGame-WaterGun/)**

## ゲームルール

- 画面内を跳ね回る**ボール**を水鉄砲で撃って消滅させるゲーム
- ボールに水を当てると**2つに分裂**する
- ボールは**3段階**で小さくなり、最小サイズのボールに当てると消滅する

| サイズ | 被弾時 |
|--------|--------|
| 大（赤） | → 中×2 に分裂 |
| 中（オレンジ） | → 小×2 に分裂 |
| 小（黄） | → 消滅！ |

- すべてのボールを消滅させると**次のステージ**へ進む
- ステージが進むごとにボールの数が1個ずつ増加（最大6個）
- **ボールがキャラクターに当たるとゲームオーバー**

## 操作方法

| キー | アクション |
|------|-----------|
| `←` `→` | キャラクターを左右に移動 |
| `Space` | 水鉄砲を発射（上方向のみ） |
| `Space` | ゲーム開始 / リトライ |
